# Dihya Backend Assets – Branding/Icons (README)

Ce dossier contient les icônes backend (SVG, PNG) : multilingue, accessibilité, versionnées, auditables, RGPD, souveraineté numérique.

- Exemples : icon-api.svg, icon-audit.svg
- Métadonnées : voir meta_icon-api.json
- Accessibilité : description alt multilingue, ARIA, tags
- Sécurité : aucun asset frontend ici, logs d’accès anonymisés
- CI/CD : tests automatiques (lint SVG, accessibilité, hash)
- Exemples d’intégration dans main.py, index.js
