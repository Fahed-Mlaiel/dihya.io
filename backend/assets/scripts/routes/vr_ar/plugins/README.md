Ce dossier contient des plugins dynamiques pour le module vr_ar : sécurité, RGPD, audit, i18n, extensibilité, hooks, tests, CI/CD.

- Exemples fournis : plugin_audit.py, plugin_rgpd.py

- Extensible via API/CLI, activation/désactivation dynamique.
