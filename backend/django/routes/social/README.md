# Social Routes

Ce module gère les routes sociales (réseaux, interactions, IA, VR, AR, etc.).

## Fonctionnalités
- Création de posts, commentaires, likes
- Modération IA
- Sécurité (CORS, JWT, audit, WAF)
- Multilingue
- REST/GraphQL
- Gestion des rôles

## Exemples
- POST /social/post/
- GET /social/feed/

## Extensible via plugins.
