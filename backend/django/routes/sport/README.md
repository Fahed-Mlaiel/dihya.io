# Sport Routes

Ce module gère les routes pour la gestion du sport (clubs, événements, IA, VR, AR, etc.).

## Fonctionnalités
- Gestion des clubs, membres, événements
- Analyse IA des performances
- Sécurité (CORS, JWT, audit, WAF)
- Multilingue
- REST/GraphQL
- Gestion des rôles

## Exemples
- POST /sport/club/
- GET /sport/events/

## Extensible via plugins.
