# Voyage Routes

Ce module gère les routes pour les voyages (réservations, IA, VR, AR, etc.).

## Fonctionnalités
- Gestion des réservations, itinéraires, IA
- Sécurité (CORS, JWT, audit, WAF)
- Multilingue
- REST/GraphQL
- Gestion des rôles

## Exemples
- POST /voyage/book/
- GET /voyage/itinerary/

## Extensible via plugins.
