# Tourisme Routes

Ce module gère les routes pour le tourisme (sites, guides, IA, VR, AR, etc.).

## Fonctionnalités
- Recherche de sites, guides, événements
- Recommandations IA
- Sécurité (CORS, JWT, audit, WAF)
- Multilingue
- REST/GraphQL
- Gestion des rôles

## Exemples
- GET /tourisme/sites/
- POST /tourisme/avis/

## Extensible via plugins.
