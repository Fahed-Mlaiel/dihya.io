# Mobile

Ce dossier contient l’ensemble des sources, assets et scripts pour les applications mobiles du projet Dihya.io.

## Structure recommandée
- `README.md` : ce fichier
- `index.js` : point d’entrée JS/TS universel (React Native, Expo, etc.)
- `main.dart` : point d’entrée Flutter (si Flutter utilisé)
- `__init__.py` : compatibilité Python
- `src/` : code source principal (screens, components, navigation, etc.)
- `assets/` : images, icônes, polices spécifiques mobile
- `services/` : services métiers (API, stockage, notifications)
- `docs/` : documentation technique mobile
- `scripts/` : scripts d’automatisation mobile

Chaque sous-dossier doit contenir un README.md et un point d’entrée si pertinent.
