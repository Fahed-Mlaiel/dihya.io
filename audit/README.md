# Dossier /audit

Ce dossier centralise tous les fichiers, scripts et rapports liés à l’audit (sécurité, RGPD, accessibilité, logs, etc.).

## Structure recommandée
- Fichiers d’audit, logs, checklists, scripts, rapports
- Sous-dossiers scripts/ et reports/ pour l’automatisation et l’archivage

Chaque fichier doit être documenté, réutilisable et conforme aux conventions du projet.
