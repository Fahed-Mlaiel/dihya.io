# Exemples d’utilisation – DevOps

```bash
./deploy.sh
```

```bash
./backup.sh
```

```bash
./restore.sh /backups/webApp_2025-06-19_12-00-00
```

## Maintenance qualité
```bash
./maintenance.sh
```

## Initialisation de données (seed)
```bash
./seed.sh
```
