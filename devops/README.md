# DevOps

Ce dossier centralise tous les scripts, configurations et pipelines CI/CD pour l’industrialisation du projet Dihya.io.

## Structure recommandée
- `README.md` : ce fichier
- `index.js` : point d’entrée JS pour outils DevOps
- `__init__.py` : compatibilité Python
- `ci/` : pipelines CI/CD (GitHub Actions, GitLab CI, etc.)
- `scripts/` : scripts d’automatisation DevOps
- `configs/` : fichiers de configuration (Docker, Kubernetes, etc.)
- `docs/` : documentation DevOps

Chaque sous-dossier doit contenir un README.md et les fichiers métiers nécessaires.
