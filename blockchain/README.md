# Blockchain

Ce dossier contient les smart contracts, scripts de déploiement, tests et documentation liés à la blockchain pour Dihya.io.

## Structure recommandée
- `README.md` : ce fichier
- `index.js` : point d’entrée JS pour outils blockchain
- `__init__.py` : compatibilité Python
- `contracts/` : smart contracts Solidity
- `scripts/` : scripts de déploiement/interactions
- `tests/` : tests automatisés
- `docs/` : documentation technique blockchain

Chaque sous-dossier doit contenir un README.md et les fichiers métiers nécessaires.
