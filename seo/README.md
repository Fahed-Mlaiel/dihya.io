# SEO

Ce dossier centralise la gestion du SEO global, sitemap, robots.txt et balises pour Dihya.io.

## Structure recommandée
- `README.md` : ce fichier
- `index.js` : point d’entrée JS
- `__init__.py` : compatibilité Python
- `sitemap/` : génération sitemap.xml
- `robots/` : gestion robots.txt
- `balises/` : templates de balises SEO
- `docs/` : documentation SEO

Chaque sous-dossier doit contenir un README.md et les fichiers métiers nécessaires.
