# RGPD

Ce dossier centralise les scripts, documents et configurations RGPD pour Dihya.io.

## Structure recommandée
- `README.md` : ce fichier
- `index.js` : point d’entrée JS
- `__init__.py` : compatibilité Python
- `scripts/` : scripts RGPD (purge, anonymisation, etc.)
- `docs/` : documentation RGPD
- `configs/` : fichiers de configuration RGPD

Chaque sous-dossier doit contenir un README.md et les fichiers métiers nécessaires.
