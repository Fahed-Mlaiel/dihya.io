# Dossier /app/utils

Ce dossier regroupe tous les utilitaires métiers de l’application (frontend, backend, scripts, automatisations, etc.).

## Structure recommandée
- Un fichier par utilitaire métier
- index.js, __init__.py pour l’export centralisé et l’industrialisation

Chaque utilitaire doit être documenté, réutilisable et conforme aux conventions du projet.
