# Dossier /app/docs/assets

Ce dossier regroupe tous les assets documentaires : diagrammes, captures d’écran, logos, illustrations, etc. Utilisez ces fichiers pour enrichir la documentation technique et utilisateur.

## Types d’assets recommandés
- diagram_architecture.svg : diagramme d’architecture technique
- screenshot_ui.png : capture d’écran illustrative
- logo_docs.svg : logo ou branding documentation
- diagramme_sequence.svg : diagramme de séquence (optionnel)
- doc_sample.pdf : exemple de document PDF (optionnel)

Chaque asset doit être clairement nommé et documenté dans ce README.
