# Dossier /app/plugins

Ce dossier centralise tous les plugins métiers, UI, analytics, CMS, Stripe, etc. Chaque plugin est isolé dans son propre dossier, avec sa config, ses assets, ses composants, ses services et sa documentation.

## Structure recommandée
- index.js : gestion globale des plugins
- __init__.py : gestion/génération Python
- plugin.config.js : config globale
- examplePlugin/ : exemple de plugin complet

Respectez la structure et les conventions pour garantir l’industrialisation et la maintenabilité.
