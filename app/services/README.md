# Dossier /app/services

Ce dossier regroupe tous les services métiers de l’application (frontend, backend, hooks, automatisations, etc.).

## Structure recommandée
- Un fichier par service métier
- index.js, __init__.py pour l’export centralisé et l’industrialisation

Chaque service doit être documenté, réutilisable et conforme aux conventions du projet.
