# Templates

Ce dossier centralise les templates globaux (emails, documents, code, etc.) pour Dihya.io.

## Structure recommandée
- `README.md` : ce fichier
- `index.js` : point d’entrée JS
- `__init__.py` : compatibilité Python
- `emails/` : templates d’emails
- `docs/` : templates de documents
- `code/` : templates de code (snippets, boilerplate)

Chaque sous-dossier doit contenir un README.md et les fichiers métiers nécessaires.
