# Dossier /assets_global

Ce dossier regroupe tous les assets partagés entre les différentes stacks du projet (frontend, backend, mobile, etc.).

## Structure recommandée
- Un sous-dossier par type d’asset (icônes, images, logos, svg, etc.)
- index.js, __init__.py pour l’export centralisé et l’industrialisation

Chaque asset doit être documenté, réutilisable et conforme aux conventions du projet.
