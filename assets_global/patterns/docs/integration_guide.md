# Guide d’intégration des patterns

Ce guide explique comment intégrer les patterns dans vos projets frontend et backend.

- **Frontend JS/React** : importez via `index.js` ou directement depuis le dossier `svg/` ou `png/`.
- **Backend Python** : utilisez `patterns.module.py` pour accéder aux patterns déclarés.
- **Astuces** : utilisez les fichiers de preview pour générer des galeries ou des sélecteurs visuels.
