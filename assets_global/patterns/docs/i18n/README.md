# i18n – Patterns Documentation

Ce dossier contient tous les fichiers de traduction (JSON) pour la documentation des patterns, couvrant toutes les langues, variantes et dialectes métiers nécessaires au projet.

## Structure
- Un fichier `i18n_xx.json` par langue/dialecte (ex : `i18n_fr.json`, `i18n_en_gb.json`, etc.)
- Chaque fichier suit la structure métier attendue pour l’internationalisation documentaire.

## Usage
Importez le fichier correspondant à la langue/dialecte souhaité pour afficher la documentation dans la langue de l’utilisateur.

## Points d’entrée
- Ajoutez un `index.js` (ou `__init__.py` pour Python) si le dossier est utilisé comme module.

## Contact
Pour toute nouvelle langue ou correction, contactez l’équipe documentation.
