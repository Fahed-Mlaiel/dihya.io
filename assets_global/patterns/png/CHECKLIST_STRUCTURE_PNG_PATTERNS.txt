# Remarques professionnelles
- Tous les fichiers d’assets PNG doivent être classés dans le dossier assets/ par secteur, usage ou thème.
- Les fichiers d’index (index.js) et d’initialisation (__init__.py si besoin pour Python) doivent être présents uniquement si le dossier est utilisé comme module ou point d’entrée.
- Les scripts de génération et d’utilitaires doivent être documentés et testés.
- Les guides doivent être adaptés à chaque langue et variante régionale pertinente pour l’industrialisation et l’accessibilité.
- Les fichiers de test sont à centraliser dans un dossier dédié (tests/) à une étape ultérieure.
