# Guide d’intégration SVG Previews

Ce guide explique comment intégrer et utiliser les SVGs de preview dans tous les contextes (web, mobile, API, plugins, etc.).

1. Importer le module d’accès JS ou Python.
2. Charger le SVG souhaité via le manifeste.
3. Utiliser les SVGs dans l’UI, la documentation, ou les assets métiers.
