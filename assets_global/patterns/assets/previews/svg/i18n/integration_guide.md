# Guide d’intégration i18n SVG Previews

Ce guide explique comment intégrer et utiliser les fichiers de traduction des SVGs de preview dans tous les contextes (web, mobile, API, plugins, etc.).

1. Importer le module d’accès JS ou Python.
2. Charger la langue souhaitée via le manifeste.
3. Utiliser les clés de traduction dans l’UI ou la logique métier.
