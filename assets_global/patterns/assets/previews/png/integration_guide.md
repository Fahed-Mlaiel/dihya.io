# Guide d’intégration PNG Previews

Ce guide explique comment intégrer et utiliser les PNGs de preview dans tous les contextes (web, mobile, API, plugins, etc.).

1. Importer le module d’accès JS ou Python.
2. Charger le PNG souhaité via le manifeste.
3. Utiliser les PNGs dans l’UI, la documentation, ou les assets métiers.
