# Guide d’intégration GIF Previews

Ce guide explique comment intégrer et utiliser les GIFs de preview dans tous les contextes (web, mobile, API, plugins, etc.).

1. Importer le module d’accès JS ou Python.
2. Charger le GIF souhaité via le manifeste.
3. Utiliser les GIFs dans l’UI, la documentation, ou les assets métiers.
