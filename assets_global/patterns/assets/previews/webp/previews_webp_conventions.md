# Conventions WEBP Previews

- Tous les WEBPs doivent être nommés `previewX.webp`.
- Les WEBPs doivent être optimisés pour le web et l’accessibilité (texte alternatif, ARIA).
- Toute contribution doit être documentée dans le changelog.
- Les métadonnées doivent être traduites dans le dossier i18n/.
