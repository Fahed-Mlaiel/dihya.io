# Guide d’intégration WEBP Previews

Ce guide explique comment intégrer et utiliser les WEBPs de preview dans tous les contextes (web, mobile, API, plugins, etc.).

1. Importer le module d’accès JS ou Python.
2. Charger le WEBP souhaité via le manifeste.
3. Utiliser les WEBPs dans l’UI, la documentation, ou les assets métiers.
