# Guide d’intégration JPG Previews

Ce guide explique comment intégrer et utiliser les JPGs de preview dans tous les contextes (web, mobile, API, plugins, etc.).

1. Importer le module d’accès JS ou Python.
2. Charger le JPG souhaité via le manifeste.
3. Utiliser les JPGs dans l’UI, la documentation, ou les assets métiers.
