# Guide d’intégration i18n assets/patterns

Ce guide explique comment intégrer et utiliser les fichiers de traduction dans les projets front-end et back-end.

1. Importer le module d’accès JS ou Python.
2. Charger la langue souhaitée via le manifeste.
3. Utiliser les clés de traduction dans l’UI ou la logique métier.
