# Guide d’intégration des assets de patterns

Ce guide explique comment intégrer les assets de patterns dans vos projets (front-end, back-end, mobile, etc.).

## Étapes principales
1. Importer les assets via le module d’accès (`index.js` ou `__init__.py`).
2. Respecter les conventions de nommage et d’organisation.
3. Consulter les exemples d’utilisation fournis.

Pour toute question, contactez l’équipe d’architecture.
