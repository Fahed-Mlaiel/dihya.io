# Exemples d’utilisation des patterns

## Utilisation en CSS
```css
body {
  background-image: url('./svg/pattern_example_1.svg');
}
```

## Utilisation en React
```jsx
<img src={require('./png/pattern_example_2.png')} alt="Pattern géométrique" />
```

## Utilisation côté backend (Python)
```python
from patterns.module import get_pattern
pattern = get_pattern('pattern_example_1')
```
