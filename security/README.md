# Security

Ce dossier centralise la sécurité globale, firewall, WAF et la gestion des accès pour Dihya.io.

## Structure recommandée
- `README.md` : ce fichier
- `index.js` : point d’entrée JS
- `__init__.py` : compatibilité Python
- `firewall/` : règles et scripts firewall/WAF
- `access/` : gestion des accès/permissions
- `docs/` : documentation sécurité

Chaque sous-dossier doit contenir un README.md et les fichiers métiers nécessaires.
