# Notification

Ce dossier regroupe les services de notification globaux (email, push, SMS, etc.) pour Dihya.io.

## Structure recommandée
- `README.md` : ce fichier
- `index.js` : point d’entrée JS
- `__init__.py` : compatibilité Python
- `services/` : logique d’envoi/gestion des notifications
- `templates/` : templates de notification
- `docs/` : documentation technique

Chaque sous-dossier doit contenir un README.md et les fichiers métiers nécessaires.
