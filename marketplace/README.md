# Marketplace

Ce dossier regroupe l’UI, l’API et la logique métier pour le marketplace de plugins Dihya.io.

## Structure recommandée
- `README.md` : ce fichier
- `index.js` : point d’entrée JS
- `__init__.py` : compatibilité Python
- `ui/` : interface utilisateur (React, Vue, etc.)
- `api/` : endpoints API marketplace
- `plugins/` : gestion des plugins
- `docs/` : documentation technique marketplace

Chaque sous-dossier doit contenir un README.md et les fichiers métiers nécessaires.
